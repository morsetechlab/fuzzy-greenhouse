## Rule Base

‡∏ä‡∏∏‡∏î‡∏Å‡∏é‡∏ó‡∏µ‡πà‡πÉ‡∏ä‡πâ‡πÉ‡∏ô‡∏£‡∏∞‡∏ö‡∏ö‡∏Ñ‡∏ß‡∏ö‡∏Ñ‡∏∏‡∏° Fuzzy Logic ‡∏ô‡∏µ‡πâ‡∏Ñ‡∏£‡∏≠‡∏ö‡∏Ñ‡∏•‡∏∏‡∏°‡∏™‡∏ñ‡∏≤‡∏ô‡∏Å‡∏≤‡∏£‡∏ì‡πå‡∏ï‡πà‡∏≤‡∏á ‡πÜ ‡∏ó‡∏µ‡πà‡∏≠‡∏≤‡∏à‡πÄ‡∏Å‡∏¥‡∏î‡∏Ç‡∏∂‡πâ‡∏ô‡πÉ‡∏ô‡πÇ‡∏£‡∏á‡πÄ‡∏£‡∏∑‡∏≠‡∏ô

```text
# üî• ‡∏Å‡∏é‡∏à‡∏≤‡∏Å‡∏≠‡∏∏‡∏ì‡∏´‡∏†‡∏π‡∏°‡∏¥‡πÅ‡∏•‡∏∞‡∏Ñ‡∏ß‡∏≤‡∏°‡∏ä‡∏∑‡πâ‡∏ô
IF Temp is Hot AND Hum is Dry         ‚Üí Fan is High   AND Mist is High
IF Temp is Hot AND Hum is Normal      ‚Üí Fan is High   AND Mist is Low
IF Temp is Hot AND Hum is Wet         ‚Üí Fan is Medium AND Mist is Off

IF Temp is Medium AND Hum is Dry      ‚Üí Fan is Medium AND Mist is Medium
IF Temp is Medium AND Hum is Normal   ‚Üí Fan is Medium AND Mist is Low
IF Temp is Medium AND Hum is Wet      ‚Üí Fan is Low    AND Mist is Off

IF Temp is Cold AND Hum is Dry        ‚Üí Fan is Off    AND Mist is Low
IF Temp is Cold AND Hum is Normal     ‚Üí Fan is Off    AND Mist is Off
IF Temp is Cold AND Hum is Wet        ‚Üí Fan is Off    AND Mist is Off

# üí° ‡∏Å‡∏é‡∏à‡∏≤‡∏Å‡πÅ‡∏™‡∏á
IF Light is Dark                      ‚Üí LED is Bright
IF Light is Normal                    ‚Üí LED is Normal
IF Light is Bright                    ‚Üí LED is Off

# üß† ‡∏Å‡∏é‡∏ú‡∏™‡∏°‡∏´‡∏•‡∏≤‡∏¢‡∏õ‡∏±‡∏à‡∏à‡∏±‡∏¢
IF Temp is Hot AND Hum is Dry AND Light is Dark ‚Üí Fan is High AND Mist is High AND LED is Bright

IF Temp is Medium AND Hum is Dry AND Light is Bright  ‚Üí Fan is Medium AND Mist is Medium AND LED is Off

IF Temp is Cold AND Light is Dark ‚Üí Fan is Off AND Mist is Off AND LED is Dim
```
